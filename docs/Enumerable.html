equal_unordered?(other) click to toggle source

Part of BareTest - require ‘baretest/assertion/support‘

Returns true if all elements of self occur the same amount of times in other, regardless of order. Uses eql? to determine equality.

Example: 

  [2,1,3,2].equal_unordered?([3,2,2,1]) # => true
